# 業檢 - 存款均額比較（活化）
### 操作步驟
- Step1: 到 Access 找到 「業檢 - 存款均額比較（活化）」這支查詢
- Step2: 添加「當前月」子查詢
    - 複製上個月的子查詢來更改即可
- Step3: 在「主查詢」中添加「當前月所需要的欄位」
    - 複製上個月的子查詢來更改即可
- Step4: 匯出「業檢 - 存款均額比較（活化）」為 Excel 檔案
